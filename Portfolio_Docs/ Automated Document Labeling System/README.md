## Introduction  

Our company's **Laravel-based web platform** required employees to manually categorize over **70 types of documents** (CNICs, passports, invoices, agreements, etc.). This manual process was **time-consuming, prone to errors, and inefficient**.  

To address this, I developed an **AI-powered solution** that integrates **Machine Learning, OCR, and Deep Learning** into Laravel. The system extracts text from **document images** via **OCR**, processes it using a **trained deep-learning model**, and instantly assigns the correct document labelâ€”**removing manual input**. The **entire pipeline** is integrated with Laravel via a **REST API**, where uploaded images are sent to a **Python-based ML model** that returns the **predicted label in real time**.
